Roadmap

Here is a roadmap of planned features in Tritonus. If you need features earlier, mail me. On the other hand, there is no guarantee.

(*) indicates features that are already present in the development version.

est. Release date

0.2
(stable)

  • reading and writing of .au, .aiff and .wav files
  • basic playback and recording
  • Alaw and mu-law converter
  • MP3 decoder (pure-Java)
  • reading of MIDI files
  • basic Sequencer
  • basic hardware Synthesizer support
  • external MIDI I/O
  • Java Sound 1.0 API

0.4
(stable)

  • MP3 encoder (*)
  • linear (PCM2PCM) converter(*)
  • sampling rate converter
  • internal floating point streaming architecture (*/2)
  • ALSA support (allows hardware mixing) (*)
  • smart selection of converters
  • support for multiple soundcards (*)
  • Mixer details
  • Line details
Perhaps:
  • full Clip support
  • Volume and balance/pan controls
  • writing of MIDI files (*)
  • handling of Sysex messages
  • Sequencer capturing
  • Sequencer and MIDI I/O on Mac and Windows95/98
  • Software Synthesizer (Linux only) (*)
  • Sun-compatible Service Provider mechanism (*)
  • Support for GNU/Linux ppc (*)
  • Split packages for easier distribution of plug-ins
