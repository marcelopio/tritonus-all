Java Sound Encoding and File Type Database

Java Sound has a flexible concept to extend the supported file types (like WAVE or AIFF) and encodings (like PCM or ulaw). Encodings and file types are referenced by unique strings as identifiers in the classes AudioFormat.Encoding and AudioFileFormat.Type, respectively.

This database is an effort to provide a central list of standardized identifiers. Everybody can add new proposals for encodings and file types. In a public process, the proposals are reviewed.
This database also provides additional information for each encoding and file type, e.g. mandatory fields or values in AudioFormat.

Currently there are encodings and file types in the database.
