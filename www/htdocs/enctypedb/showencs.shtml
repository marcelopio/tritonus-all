Java Sound Encoding Database

This encoding database is the effort to standardize the encoding names (IDs) for use with Java Sound. In a publicly discussed process, encoding proposals are reviewed and may be accepted as a standard encoding.
Encodings are used in AudioFormat to specify the format and encoding of byte streams (e.g. AudioInputStream).

Everybody can make a proposal for a new encoding.

Click on a encoding ID to see detailed information.
