Java Sound File Type Database

This database is the effort to standardize the file type names (IDs) and their extensions for use with Java Sound. In a publicly discussed process, file type proposals are reviewed and may be accepted as a standard file type.
File types are used in AudioFileFormat to specify the type and extension of a file.

Everybody can make a proposal for a new file type.

Click on a file type ID to see detailed information.
