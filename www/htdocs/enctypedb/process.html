Java Sound File Type and Encoding Database

The process of adding a new file type or encoding:

  1. Upon submission of a new file type or endoding, it will be added to the database with status proposal.
  2. The proposal will be mailed to the maintainers of the database, Florian and Matthias, and the tritonus-user mailing list. The status will then change to in review and the new encoding is discussed. In this phase, the person who submitted the proposal may be contacted for clarifications.
  3. When a consent is found, its status will change to accepted or not accepted.
  4. Accepted file types and encodings can then be used in Java Sound programs.
