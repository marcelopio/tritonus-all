Tritonus: Mailing Lists

tritonus-user This list is for questions on installation, setting up and using Tritonus. Requests for new features should be placed here, too.

tritonus-devel This list is for communication amoung the developers of Tritonus.

tritonus-announce This list is for announcements of new versions of Tritonus, major web site updates and other important project reorganisations. This list is moderated (postings have to be approved by the list administrator).

tritonus-commits This list is for developers. It sends one e-mail per CVS commit.
